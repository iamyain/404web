Kegiatanku
Sedang mengikuti program Bangkit dengan learning path cloud computing. Program Bangkit Kampus Merdeka adalah program pelatihan teknologi yang diselenggarakan oleh Google dan Kementerian Pendidikan dan Kebudayaan Indonesia. Program ini ditujukan untuk mahasiswa/i dan lulusan baru yang ingin memperdalam pengetahuan dan keterampilan di bidang teknologi seperti machine learning, mobile development, cloud computing, dan data analytics. Selama program ini, peserta akan mengikuti kursus online, proyek praktikum, dan pembelajaran berkelompok, serta mendapatkan dukungan dari mentor profesional. Program Bangkit Kampus Merdeka bertujuan untuk membantu menciptakan lebih banyak talenta teknologi yang berkualitas dan siap berkontribusi pada industri teknologi di Indonesia.
